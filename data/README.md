## Primary Dataset: Online Retail UCI

**Dataset**: Online Retail (UCI Machine Learning Repository)
**URL**: https://archive.ics.uci.edu/dataset/352/online+retail
**License**: CC BY 4.0 — Free for research and educational use. **Not for redistribution of raw data.**
**Note**: This repo includes **only synthetic sample data** for demonstration. The raw dataset is not committed.

### Download Instructions

1. Visit https://archive.ics.uci.edu/dataset/352/online+retail
2. Download `Online Retail.xlsx`
3. Place it at `data/raw/online_retail.xlsx`
4. Run `make etl` to process it

### Schema (Online Retail UCI)

| Column       | Type     | Description                                   |
|-------------|----------|-----------------------------------------------|
| InvoiceNo   | string   | Invoice number; starts with 'C' if cancelled  |
| StockCode   | string   | Product/item code                             |
| Description | string   | Product name                                  |
| Quantity    | int      | Number of units per transaction               |
| InvoiceDate | datetime | Date and time of invoice                      |
| UnitPrice   | float    | Unit price in GBP (£)                         |
| CustomerID  | float    | 5-digit customer identifier (nullable)        |
| Country     | string   | Country where customer resides                |

## Synthetic Sample Data

`data/sample/sample_transactions.csv` is generated by `scripts/generate_sample.py` with:
- 50 synthetic customers (C1001–C1050)
- 80 synthetic products (P001–P080)
- ~1,500 transaction rows across 12 months (2023)
- Realistic distributions for RFM, cohort, and MBA analysis

Regenerate: `make sample-data`
