# Data Card

## Primary Dataset: Online Retail UCI

| Field | Value |
|-------|-------|
| Source | UC Irvine Machine Learning Repository |
| URL | https://archive.ics.uci.edu/dataset/352/online+retail |
| License | CC BY 4.0 |
| Time Range | 2010-12-01 to 2011-12-09 |
| Format | XLSX / CSV |
| Rows | ~541,000 transactions |
| Geography | UK-based online gift shop, customers worldwide |

> **Repository Policy**: Raw data is not committed. This repo uses synthetic demo data only.

## Schema

| Column | Type | Nullable | Description |
|--------|------|----------|-------------|
| InvoiceNo | VARCHAR | N | Invoice number; starts with 'C' if cancelled |
| StockCode | VARCHAR | N | Product/item code (5-7 alphanumeric chars) |
| Description | VARCHAR | Y | Product name (free text) |
| Quantity | INTEGER | N | Units per transaction; negative = return |
| InvoiceDate | DATETIME | N | Transaction timestamp |
| UnitPrice | FLOAT | N | Price per unit in GBP (£) |
| CustomerID | FLOAT | Y | 5-digit customer ID (null = guest checkout) |
| Country | VARCHAR | N | Customer's country |

## Data Quality Issues (Handled by `preprocessing/cleaner.py`)

| Issue | Frequency | Handling |
|-------|-----------|---------|
| Null CustomerID (guest checkout) | ~25% of rows | Dropped — can't build customer profile |
| Cancelled orders (InvoiceNo starts 'C') | ~2% | Dropped — not completed purchases |
| Negative Quantity (returns/adjustments) | ~2% | Dropped |
| Zero UnitPrice | ~1% | Dropped |
| Non-product StockCodes (POST, DOT, M, etc.) | Rare | Dropped via code format validation |

After cleaning: ~77% of raw rows are retained.

## Synthetic Demo Data

`data/sample/sample_transactions.csv` is generated by `scripts/generate_sample.py` with:

| Property | Value |
|----------|-------|
| Rows | ~1,500 |
| Customers | 50 (C1001–C1050) |
| Products | 80 (P001–P080) |
| Date range | 2023-01-01 to 2023-12-31 |
| Countries | UK (78%), DE/FR/ES/NL/BE/AU (22%) |
| Random seed | 42 (reproducible) |

Product descriptions use realistic gift-shop naming conventions (matching UCI dataset style). Prices follow a log-normal distribution (£0.50–£50).

**MBA-realistic pattern**: 15 product bundles are hard-coded so that Apriori finds real associations (e.g., candles + holders are seeded to co-occur at 40% probability).
